# Transparent DAO Treasury with Security Controls

## Project Goal
Create a DAO where funds are managed via voting, but with extra safety layers like Time-locks, a Security Circuit Breaker, and Risk Scoring.

## Tech Stack
- **Smart Contracts**: Solidity (Hardhat, OpenZeppelin v5.0+)
- **Frontend**: Next.js (App Router), Tailwind CSS, Shadcn UI
- **Blockchain Interaction**: Wagmi, Viem, ConnectKit
- **Network**: Base Sepolia Testnet

## Structure
- `/contracts`: Smart contracts, tests, and deployment scripts.
- `/frontend`: Next.js web application.
